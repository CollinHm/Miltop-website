# Leeruitkomsten per Focusgebied

| Focusgebied           | Leeruitkomsten                                                                                                    | Werkproces   |
|-----------------------|-------------------------------------------------------------------------------------------------------------------|--------------|
| Voorbereiding         | **Stemt opdracht af, plant werkzaamheden en bewaakt de voortgang:**                                               | B1-K1-W1     |
|                       | * Je stemt het Functional Design Document (FDD) af met betrokken personen en past het zonodig aan.                |              |
|                       | * Je stelt taken op, op basis van het TDD.                                                                        |              |
|                       | * Je maakt een sprintplanning op basis van je backlog.                                                            |              |
|                       | * Je houdt standups en past de planning zonodig aan.                                                              |              |
|                       | **Maakt een technisch ontwerp voor software:**                                                                    | B1-K1-W2     |
|                       | * Je maakt een technical design document (TDD) op basis van een aangeleverd FDD.                                  |              |
| Realisatie            | **Realiseert (onderdelen van) software:**                                                                         | B1-K1-W3     |
|                       | * Je realiseert software die werkt en voldoet aan de opdracht, het ontwerp en de hierbij geldende codeconventies. |              |
|                       | * Je kunt je code op regelniveau uitleggen en gebruikt daarvoor de juiste terminologie.                           |              |
| Testen                | **Test software:**                                                                                                | B1-K1-W4     |
|                       | * Je kiest een passende testmethode en voert deze correct uit.                                                    |              |
|                       | * Je trekt logische conclusies op basis van de testresultaten.                                                    |              |
|                       | **Doet verbetervoorstellen voor de software:**                                                                    | B1-K1-W5     |
|                       | * Je vertaalt de eisen/wensen vanuit de review met de klant naar concrete backlog items.                          |              |
|                       | * Je vertaalt de testresultaten naar concrete backlog items.                                                      |              |
| Professionele handelen| **Werkt samen in een projectteam:**                                                                               | B1-K2-W1     |
|                       | * Je houdt je aan de gemaakte werkafspraken.                                                                      |              |
|                       | * Je communiceert over en vraagt feedback op je eigen werk.                                                       |              |
|                       | * Je maakt je werk en proces inzichtelijk en je beargumenteert je keuzes.                                         |              |
|                       | **Presenteert het opgeleverde werk:**                                                                             | B1-K2-W2     |
|                       | * Je informeert betrokken personen over de voortgang.                                                             |              |
|                       | **Evalueert de samenwerking:**                                                                                    | B1-K2-W3     |
|                       | * Je benoemt wat goed ging en beter kan aan je eigen functioneren, prestaties en het teamproces.                  |              |
